## Codebase Patterns
- Dashboard is a Vite SPA with pages in `dashboard/src/pages/` and components in `dashboard/src/components/`
- Router is in `dashboard/src/router.tsx` using react-router-dom v7
- Pre-existing TS errors in `server/routes/archives.ts` and `server/routes/projects.ts` (string | string[] type issues) — not introduced by this work
- Sidebar nav items are defined in `dashboard/src/components/sidebar.tsx` as a `navItems` array
- Project uses Tailwind v4 with CSS variables in `dashboard/src/globals.css`
- WebSocket provider wraps dashboard layout in router.tsx
- Dashboard source is at `dashboard/` from repo root (NOT under `scripts/ralph/dashboard/`)
- Design system reference is at `design-system/MASTER.md` from repo root
- RalphStatusIndicator supports `compact` prop for icon-only display
