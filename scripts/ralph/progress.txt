# Ralph Progress Log
Started: 2026-03-02

## Codebase Patterns
- Next.js 16 App Router with Tailwind CSS v4 + shadcn/ui in `dashboard/` directory
- Dark theme enforced via `className="dark"` on `<html>` element
- Sidebar layout uses `(dashboard)` route group with shared layout.tsx
- shadcn/ui components installed via `npx shadcn@latest add <component>`
- Sidebar colors: zinc-950 background, zinc-800 selected item, zinc-400 text
- All pages are at `app/(dashboard)/*/page.tsx`
- All UI text is in Chinese (zh-CN localization)
- WebSocket provider wraps all dashboard pages via layout.tsx
- Zustand store at lib/store.ts manages global state (prd, ralphStatus, wsState)
- API routes at app/api/* handle prd CRUD, ralph start/stop, logs, archives, config
- Terminal uses xterm.js via dynamic import (no SSR)
- Kanban uses @hello-pangea/dnd for drag-and-drop
- Toast notifications via sonner
- Tailwind CSS v4 uses @theme inline and CSS variables in globals.css
- CSS custom properties defined in :root and .dark blocks
- All components use Chinese text labels (zh-CN)
- Brand colors: --brand-cyan (#06B6D4) and --brand-blue (#3B82F6) defined in globals.css
- Utility classes card-glow, gradient-border, skeleton available in globals.css
- Animation classes: animate-fade-in, animate-slide-up available globally
- Sidebar NavContent is shared between desktop aside and mobile Sheet (update once, both updated)
- Use `style={{ borderImage: '...' }}` for gradient borders on elements that need non-Tailwind gradients
- Active nav items use `relative` positioning with an absolutely-positioned gradient bar child
